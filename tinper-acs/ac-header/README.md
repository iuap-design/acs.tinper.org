## 依赖

- react >= 15.3.0
- react-dom >= 15.3.0
- prop-types >= 15.6.0

## 使用方法

```js
import AcHeader from 'ac-header';
import 'ac-header/build/AcHeader.css';

```



## API

|参数|说明|类型|默认值|
|:---|:-----|:----|:------|
|className|类名|string|'ac-header'|
|icon|图标内容|any|-|
|title|标题|string|-|
|showBack|是否显示返回|bool|-|
|backClick|返回按钮点击回调函数|func|-|

#### 开发调试

```sh
$ npm install -g bee-tools
```
