## API 


参数 | 类型 | 默认值 | 是否必填 | 说明
--------- | ---- | ------|----------- | -----------------
showHeader | `bool` | | 否 | 是否显示导航栏
locale | `string` | `zh` | 否 | 组件语言类型，默认是简体中文，目前支持英文(en或en_US)，简体中文(zh_CN)，繁体中文(zh_TW)
