 ## API

|参数|说明|类型|默认值|
|:---|:-----|:----|:------|
|title|标题|node|-|
|search|查询回调|func|-|
|reset|清空回调|func|-|
|Children|子元素：Complex/Sample 组件|node|-|
|hasChose|是否可以选择查询方案|node|-|
|localeCookie|获取多语cookie的key值|string|`locale`|

### FormItem

|参数|说明|类型|默认值|
|:---|:-----|:----|:------|
|label|查询表单元素标签|string|-|
|required|查询表单元素是否必填|bool|-|
|tooltip|鼠标enter时显示内容|string|-|
