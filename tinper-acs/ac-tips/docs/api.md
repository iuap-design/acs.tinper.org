## API

### AcTips.create

|参数|说明|类型|默认值|
|:---|:-----|:----|:------|
|type|提示类型 `success`、`error`、`warning`|string|`success`|
|content|提示内容|node|-|
|top|到页面顶端的距离|number|50|
|duration|显示多久自动隐藏，error不隐藏，单位毫秒|number|5000|
|zIndex|提示的层级|number|9|
