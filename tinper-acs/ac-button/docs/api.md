# 按钮 Button

## 何时使用

代替原生按钮

## 如何使用

```
$ ynpm install @yonyou/ac-button --save

引入

import AcButton from '@yonyou/ac-button';

样式

import '@yonyou/ac-button/dist/index.css';

```

## API 

|参数|说明|类型|默认值|
|:--|:---:|:--:|---:|
