 ## API

|参数|说明|类型|默认值|
|:---|:-----|:----|:------|
|search|查询回调|func|-|
|reset|清空回调|func|-|
|Sample|简单查询内容|node|-|
|Complex|复杂查询内容|node|-|
|FormItem|查询表单元素内容|node|-|
|FormItem.label|查询表单元素标签|string|-|
|FormItem.required|查询表单元素是否必填|bool|-|
