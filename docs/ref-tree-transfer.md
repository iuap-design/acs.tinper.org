## 树形参照分类为下列三种

### RefTreeTransfer
    
    参照弹出窗，没有输入框，使用时可根据自己需要定义具体的文本框。

### RefTreeTransferWithInput
    
    带文本框的参照弹出窗。在 RefTreeTransfer 基础上封装实现。

### createRefTreeTransfer
    
    非 ReactJS 调用方式，与 RefTreeTransfer 相同没有输入框，使用时可根据自己需要定义具体的文本框。

## RefTreeTableWithInput 增量 API
